Procedural Generation

This project was built during my 3rd year at Coventry University and features different elements such as generated terrain, water, clouds and trees. I wanted to push this project beyond module's requirements and explored other algorithms, for example, L-systems used for tree generation.

I really enjoyed making this project and I have learnt many graphics techniques and shader development. Procedural Generation is the main motivation for my dessertation project which is coming soon.
