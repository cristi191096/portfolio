A raytracing algorithm

Ray Tracer

By doing this project I had the chance to demonstrate a vast knowledge of computer graphics and learn how data structures such as Trees can be used for optimisation. This is my first attempt to a multi-threaded application as well. I believe there are many ways to improve the project. I would add the GUI menu for a better experience, I would add different ways of partitioning the screen like Binary Space Partitioning (BSP), I would improve the lighting effect, for example surface scattering.

I have enjoyed this project and I have learned how to manage my time in order for me to deliver the best work I possibly could.
